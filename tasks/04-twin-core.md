# Task 04 - Twin Core

## Objetivo

Implementar o nucleo do gêmeo digital com modelo de estado e motor inicial de decisao.

## Entradas

* `/docs/componentes.md`
* `/docs/arquitetura.md`
* `/docs/metricas.md`

## Entregas

* modelo de estado operacional;
* consumo de metricas/eventos;
* calculo de indicadores de saude;
* regras iniciais de deteccao;
* classificacao de risco;
* registro de decisoes.

## Fora de escopo

* machine learning avancado;
* otimizacao adaptativa complexa.

## Criterio de pronto

O gêmeo consegue refletir o estado do sistema e emitir alertas coerentes.
