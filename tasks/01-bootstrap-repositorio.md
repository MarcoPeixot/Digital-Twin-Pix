# Task 01 - Bootstrap Repositorio

## Objetivo

Criar a fundacao tecnica do repositorio do MVP.

## Entradas

* `/docs/arquitetura.md`
* `/docs/componentes.md`

## Entregas

* estrutura inicial de pastas;
* README inicial;
* `docker-compose.yml` minimo;
* arquivos `.env.example`;
* servico placeholder de cada modulo principal;
* instrucoes locais de execucao.

## Fora de escopo

* logica detalhada de negocio;
* cenarios completos;
* mitigacao avancada.

## Criterio de pronto

O repositorio sobe localmente e todos os servicos placeholders iniciam sem erro.
