# Executar Experimentos

Leia:

* `/docs/cenarios-experimentais.md`
* `/docs/metricas.md`
* `/docs/resultados.md`

Implemente a infraestrutura minima para executar experimentos reproduziveis.

Entregue:

* scripts de execucao por cenario;
* arquivos de configuracao parametrizados;
* coleta de metricas;
* persistencia dos resultados;
* instrucoes para rodar com e sem gêmeo digital.

Valide especialmente:

* reprodutibilidade;
* nomes consistentes de metricas;
* comparabilidade entre cenarios;
* registro de versao/configuracao.
